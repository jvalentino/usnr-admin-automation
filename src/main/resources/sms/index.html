Mass Message Sender

This tool can be used to send email and text messages to any individual or group of individuals that is defined in Manual Unit Inputs.xlsx.

This tool evolved out of a need to quickly make content with unit members, in which text messaging proved to be most effective means. Later the input was changed to take the spreadsheet used in the Report Merger application known as Manual Unit Inputs.xlsx, which contains first name, last name, rank/rate, and cell #, and any number of email addresses. Later the Report Merger application had SMS abilities added to it using the services written for this application, and then email abilities. The SMS and Email functionality was unified and made into a series of reusable components and services.

Limitations

  • IP addresses are limited to 75 texts per day. Phone numbers are limited to 3 texts every 3 minutes.
  • Some carriers may deliver text messages from txt@textbelt.com
  • People cannot reply back to the number that sends the SMS, make sure to provide your contact number
